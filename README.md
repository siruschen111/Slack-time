# 🐟Slack time
隐形阅读助手 | Windows 专用 | 极速丝滑版  这是一个桌面隐形阅读器。它没有边框，背景完全透明，文字直接“浮”在桌面上，像幽灵一样低调。配合强大的搜索和极速滚动功能，助你在繁忙的工作间隙安全、高效地享受阅读时光。
✨ 核心亮点
👻 极致隐身，安全第一
全透明背景：采用 Windows 特殊透明技术，消除文字边缘杂色，只留纯净的灰色文字浮于桌面。
无边框设计：去除了所有显眼的窗口标题栏和边框，完美融入背景软件（如 IDE 或文档）。

低调配色：默认深灰色字体，看起来就像屏幕上的水印或注释。

🎮 实体操控，极速浏览
顶部拖拽条：窗口顶部设有 10px 高的深色实体条，解决透明窗口难以抓取的问题，按住即可随意拖动。

自由缩放：右下角设有 ◢ 手柄，按住可自由拉伸窗口大小。

🚀 极速滚动：重写了底层滚动逻辑。鼠标滚轮速度提升 3 倍，键盘方向键直接控制视窗滑动，带来浏览器般的丝滑体验。

🔍 强大的搜索导航系统
Ctrl+F 查找：支持全文关键词搜索。

双色高亮：

🟨 黄色：标记所有匹配内容。

🟧 橙色：醒目标记当前阅读位置。

F3 循环跳转：一键自动定位到下一个关键词，无需手动翻找。

🛡️ 安全机制
只读模式：屏蔽键盘输入，防止误删小说内容。

Esc 紧急应对：一键清除高亮恢复低调；再按一键直接退出程序。

📖 使用指南
1. 启动与加载
运行程序，屏幕会出现一段灰色的初始化提示。

在文字区域或顶部横条上 点击鼠标右键。

选择菜单中的 📂 打开小说 (.txt)（程序会自动识别 UTF-8 或 GBK 编码）。

2. 窗口调整
移动位置：鼠标按住窗口 最顶部 的深灰色细条并拖动。

改变大小：鼠标按住窗口 右下角 的 ◢ 符号并拖动。

快速退出：双击顶部的深灰色细条即可关闭程序。

3. 搜索与阅读
滚动浏览：使用鼠标滚轮或键盘 ↑ ↓ 键享受极速滚动；使用 PageUp/PageDown 快速翻页。

查找内容：按下 Ctrl + F，输入关键词。

跳转下一个：搜索后，按下 F3 键，视图会自动跳转到下一处匹配位置。

恢复隐蔽：阅读完毕后，按下 Esc 键清除所有显眼的高亮颜色。
